# Writing Voice Skill

Use mcp-me to write authentically in the user's voice.

## When to activate

- User asks to write a post, article, thread, newsletter, or bio "in my voice"
- User asks to rewrite text to match their style
- User asks how they write or what their tone is

## MCP tools to use

| Tool / Resource | Purpose |
|-----------------|---------|
| `me://writing/style` | Format profiles and tone definitions |
| `me://writing/corpus` | List of published texts (metadata) |
| `me://writing/samples?profile=` | Excerpt samples for a profile |
| `analyze_writing_style` | Stats: sentence length, vocabulary, punctuation |
| `search_writing_corpus` | Find texts by keyword |
| `get_writing_references` | Relevant excerpts for a topic |
| `emulate_my_voice` | Prompt to generate in user's voice |
| `rewrite_in_my_voice` | Prompt to rewrite existing text |

## Format profiles

- **personal_blog** — conversational, first-person, longer form
- **tech_news** — informative, neutral, concise
- **ironic_thread** — witty, short, social media tone

## Always combine with profile context

Writing is one dimension of identity. Also read career, skills, projects, and personality for factual accuracy.
